# 🛒 E-commerce Analytics Dashboard & Business Intelligence

## Overview

This project explores an e-commerce dataset using SQL and Python to uncover business insights related to sales performance, customer behaviour, product ratings, pricing, discounts, and inventory.

The objective was to transform raw transactional data into actionable business intelligence through exploratory analysis and data visualisation.

---

## Objectives

- Analyse sales performance
- Understand customer ratings
- Evaluate pricing and discount strategies
- Identify top-performing product categories
- Explore inventory availability
- Generate business recommendations using data
