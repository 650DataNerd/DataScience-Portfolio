# 👩 Gender Pay Gap Analysis

## Overview

This project investigates wage disparities between genders using statistical analysis and data visualization techniques. The analysis explores income distribution, wage trends over time, and gender-based differences to better understand patterns of inequality within the dataset.

The goal of this project was to apply exploratory data analysis (EDA) and statistical techniques to transform raw salary data into meaningful insights that can support evidence-based discussions around workplace equity.

---

## Objectives

- Explore wage distribution across genders
- Analyze income trends over time
- Visualize salary disparities
- Identify potential indicators of gender pay inequality
- Demonstrate statistical analysis and data visualization techniques
